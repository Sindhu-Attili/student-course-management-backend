# Student Course Management System Backend

A Learning Management System (LMS) backend built using Django, Django REST Framework, and MySQL.

## Features

- JWT Authentication
- Student Management
- Course Management
- Student Enrollments
- Payment Tracking
- Assignment Management
- Assignment Submissions
- Automatic Course Completion Tracking
- Certificate Generation
- Swagger API Documentation

## Tech Stack

- Python
- Django
- Django REST Framework
- MySQL
- JWT Authentication
- Swagger/OpenAPI

## Installation

```bash
git clone <repository-url>
cd student-course-management-backend

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
